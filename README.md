#  Wine Quality Analysis - EDA Project

Bu proje, şarap kalitesini etkileyen kimyasal özellikleri keşfetmek amacıyla yapılmıştır.

##  Proje Önemli Noktaları (Highlights)
* **Otomatik Görselleştirme:** 12 farklı değişkenin dağılımı `enumerate` ve `subplots` kullanılarak tek bir döngüde görselleştirildi.
* **Gelişmiş Heatmap:** Korelasyon matrisi, gereksiz tekrarları önlemek için NumPy maskeleme yöntemiyle profesyonelce sunuldu.
* **İstatistiksel İçgörü:** Alkol oranı ile kalite arasındaki pozitif ilişki (0.48) Box Plotlar ile kanıtlandı.

## 🛠 Kullanılan Teknolojiler
* Python (Pandas, NumPy)
* Seaborn & Matplotlib (Görselleştirme)
